JavaScript - IV/03 - Les valeurs de retours

Return

1- Définitions

Certaines fonctions prédéfinies vont renvoyer une valeur de retour tandis que d'autres ne vont pas en renvoyer.

L'instruction return met fin à l'exécution d'une fonction, ce qui signifie que toutes les autres opérations qui suivent return seront ignorées !
