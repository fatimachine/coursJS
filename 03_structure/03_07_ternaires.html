Les structures conditionnelle ternaires

Les ternaires sont une autre façon d'écrire nos conditions en utilisant une syntaxe basée sur l'opérateur ternaire ?:.

1- Ecriture

On peut écrire la condition sur une seule ligne

Ex. document.getElementById('p4').innerHTML = y > 10 ? 'y est supérieur à 10 ' : 'y est inférieur à 10';

les ternaires se présentent sous la forme suivante : test ? code à exécuter si true : code à exécuter sinon
