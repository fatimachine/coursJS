1- If... else

Si... sinon

La structure conditionnelle if... else sera plus complète car on va exécuter un bloc de code si un test renvoie true ou un autre dans le cas contraire.

if(x > 1){
//si le test est vrai j'exécute le bloc qui suit
alert('IF x contient ' + x + ' et cette valeur est supérieure à 1');
}else{
// sinon j'exécute celui-ci
alert('ELSE x contient ' + x + ' et cette valeur est inférieure à 1');
}
