Javascript /III-If,else

1- if...else,si...sinon

titre

La structure conditionnelle if...else sera plus complète car on va exécuter un bloc de code si un test renvoie trueou un autre dans le cas contraire.

if(x>1) {
// si le test est vrai j'exécute le bloc qui suit
alert('x contient' + x+ 'et cette valeur est supérieur à 1');
}else{
// sinon j'exécute celui-ci
alert('x contient' + x+ 'et cette valeur est inférieur à 1');
}
