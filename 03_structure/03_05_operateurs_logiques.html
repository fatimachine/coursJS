Les opérateurs logiques

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1- Présentation & utilisation des opérateurs logiques

2- Récap

Opérateurs logiques
Nom Symbole Description
AND (et) && renvoie true si toutes les comparaisons sont vraies ou false sinon.
OR (ou) || renvoie true si au moins l'une des comparaisons est vraie, ou false sinon.
NO (non) ! renvoie true si une comparaison est évalué à false, et vice versa
