JavaScript - III/01 - Structures de contrôle

Il existe 2 types de structures de contrôle ...

Les conditions & les boucles

Il existe deux types de structures de contrôle de base, que l'on retrouve dans TOUS les langages de programmation, et donc en JS.

A -Les conditions

Les conditions vont nous permettre d'éxécuter un certain nombre d'instructions si et seulement si une certaine condition est vérifiée.

ex : Si il pleut, je ne sors pas

B- Les boucles

Les boucles vont nous permettre d'éxécuter un bloc de condition en boucle tant qu'une condition donnée est vérifiée.

ex. : Dès que j'ai fait monter les 25 élèves de ma classe dans le bus, je pars.

C- Opérateurs de comparaison

Comment vérifier les conditions ?

Dans le cas où la comparaison est vérifié, le JS renvoie le booléen true, dans le cas contraire, le booléen false est renvoyé.

Le JS va automatiquement comparer la valeur à gauche de l'opérateur à celle de droite selon l'opérateur de comparaison.
