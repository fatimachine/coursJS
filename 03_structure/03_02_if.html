JavaScript - III/02 - if

- La condition if est la plus utilisée des conditions

If

1- if...

La condition if est l'une des plus utilisée et est également la plus simple à appréhender puisqu'elle va nous permettre d'exécuter un bloc de code si ET seulement si le résultat vaut true.

2- Attention ! : exceptions

Tout test d'une condition va être évalué dans un contexte booléen ; donc à true à l'exception des valeurs suivantes, qui vont être évaluées à false.

  • La valeur 0 (zéro)
  • le booléen false
  • une chaîne de caractère vide
  • la valeur null
  • la valeur undefined
  • la valeur NaN (Not a Number)

3- Inverser la valeur logique d'un test

Le code placé dans le if n'est exécuté que si le résultat de la comparaison est true.

Dans certaines situations, on préfère vérifier si la comparaison est false : on peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false.
